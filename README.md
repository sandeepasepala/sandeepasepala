# Professional Cybersecurity & Full-Stack Portfolio

A high-performance, visually stunning portfolio website for **Sandeepa Sepala**, a Cybersecurity Specialist and Full-Stack Developer. This project showcases advanced frontend engineering, security-focused UI/UX, and professional achievements in penetration testing and system design.

## 🚀 Key Features

- **Advanced Glassmorphism UI:** Modern, translucent navigation and card components with backdrop-blur effects.
- **Dynamic Particle System:** custom-built JavaScript particle engine for a professional "cyber" background aesthetic.
- **Professional Project Showcases:** Interactive project cards with integrated call-to-action buttons and live repository links.
- **Specialized Animations:**
  - **Sparkle Proficiency Bars:** Unique human language status bars that "fill" with dynamic particle sparkles.
  - **Scroll-Triggered Reveals:** Smooth intersection-observer based animations for content entry.
  - **Glow & Shimmer Effects:** Refined Tailwind-based keyframe animations for high-impact visual feedback.
- **Credential Integration:** Dedicated sections for Global Rankings (TryHackMe, HTB, HackerOne) and Cisco Netacad digital badges.
- **Mobile Responsive:** Fully optimized for all screen sizes from mobile to ultra-wide displays.

## 🛠️ Tech Stack

- **Frontend:** HTML5, JavaScript (ES6+)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/) (JIT CDN)
- **Icons:** SVG-based iconography for maximum performance and clarity.
- **Animations:** Tailwind Extended Keyframes & custom CSS.

## 📂 Project Structure

```bash
├── index.html      # Main portfolio structure and Tailwind configuration
├── index.js        # Logic for particles, sparkles, and scroll animations
├── README.md       # Project documentation
└── scripts/        # Additional utility scripts
```

## 💻 Local Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/sandeepasepala/sandeepasepala.github.io.git
   ```
2. **Navigate to the directory:**
   ```bash
   cd sandeepasepala
   ```
3. **Open in Browser:**
   Simply open `index.html` in any modern web browser. No build steps or dependencies required.
